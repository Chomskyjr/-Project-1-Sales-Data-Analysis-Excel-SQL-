# 🛒 Sales Data Analysis  

## 📌 Project Overview  
This project analyzes retail sales data to identify trends, top-selling products, and customer behavior using **Excel and SQL**.  

## 🎯 Objectives  
- Clean and prepare raw sales data.  
- Use **Excel** for descriptive analysis (pivot tables, charts).  
- Use **SQL** queries for deeper insights.  
- Provide actionable recommendations for business growth.  

## 🛠️ Tools & Technologies  
- Excel (Power Query, Pivot Tables, Charts)  
- SQL (MySQL / PostgreSQL / SQLite)  

## 📊 Steps Performed  
1. Data cleaning in Excel (missing values, duplicates, date formatting).  
2. Pivot table analysis for sales by product, region, and month.  
3. SQL queries to answer business questions (e.g., highest-selling product, top regions).  
4. Visualized monthly sales trends in Excel.  

## 🔑 Key Insights  
- 📈 Sales peaked in **December (holiday season)**.  
- 🏆 **Electronics** category generated the highest revenue.  
- 👥 Repeat customers accounted for **40%** of total purchases.
